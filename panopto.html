Panopto Custom Speed


Speed up or slow down your Panopto videos easily.
Add to Chrome

Features

Easy Setup

Install and activate the extension with just a click.

Custom Speed Options

Select which speed you want to use, and whether you want the speed to be saved for the next video you watch.

User-Friendly Interface

A simple slider to change the speeds, but also allowing you to enter a specific speed if you want.
